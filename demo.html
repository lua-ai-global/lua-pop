LuaPop Demo

Customize the chat widget configuration below and see it update in real-time.

Widget Configuration

// Add this to your website
<link rel="stylesheet" href="https://lua-ai-global.github.io/lua-popup-demo/style.css">
<script src="https://lua-ai-global.github.io/lua-popup-demo/lua-pop.umd.js"></script>
<script>
  // Initialize the widget with your configuration
  window.LuaPop.init({
    authToken: "YOUR_API_TOKEN",
    position: "bottom-right", // bottom-right, bottom-left, top-right, top-left
    theme: "light", // light or dark
    buttonText: "Chat with us",
  });
</script>
